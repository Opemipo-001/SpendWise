# SpendWise

SpendWise is a smart expense tracking web application built with Django that helps users manage their income, expenses, savings, and spending habits through an interactive dashboard.

---

## Features

- User authentication (Register/Login/Logout)
- Add income and expense transactions
- Edit and delete transactions
- Expense categorization
- Dashboard analytics
- Monthly income vs expense tracking
- Pie chart visualization of expenses
- Smart budget alert system
- Savings estimation
- Responsive Bootstrap UI

---

## Tech Stack

- Python
- Django
- SQLite
- Bootstrap 5
- Chart.js

---

## Smart Budget Logic

SpendWise includes a budget monitoring system that alerts users when their expenses exceed 75% of their total income.

Example:
- Income = ₦400,000
- Recommended spending limit = ₦300,000
- Alert triggers when expenses exceed ₦300,000
